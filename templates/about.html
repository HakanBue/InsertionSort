Insertion Sort

Insertion Sort ist ein einfaches und intuitives Sortierverfahren, das besonders bei kleinen Datensätzen effizient arbeitet. Der Algorithmus funktioniert nach dem Prinzip des Kartensortierens: Er beginnt mit dem zweiten Element im Array, da ein einzelnes Element per Definition sortiert ist, und vergleicht dieses Element mit den vorhergehenden, um es in die richtige Position innerhalb des bereits sortierten Teils der Liste einzufügen. Wenn das aktuelle Element kleiner ist als das vorherige, werden die vorherigen Elemente um eine Position nach rechts verschoben, um Platz für das aktuelle Element zu schaffen. Dieser Vorgang wird wiederholt, bis das gesamte Array sortiert ist. Beispielsweise wird ein Array wie [5, 2, 4, 6, 1, 3] Schritt für Schritt in [1, 2, 3, 4, 5, 6] umgewandelt. Insertion Sort hat im schlechtesten Fall eine Zeitkomplexität von O(n²) und im besten Fall von O(n), wenn das Array bereits sortiert ist. Der Algorithmus ist stabil, da gleiche Elemente in ihrer ursprünglichen Reihenfolge bleiben, und arbeitet In-Place, was bedeutet, dass er nur eine konstante Menge an zusätzlichem Speicher benötigt. Aufgrund seiner Effizienz bei kleinen Arrays wird Insertion Sort oft in Kombination mit anderen Sortieralgorithmen verwendet und findet Anwendung in Echtzeit-Datensortierungen oder Situationen, in denen nur wenige Elemente sortiert werden müssen.
